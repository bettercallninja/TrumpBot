📋 TrumpBot v2.0.0 Enterprise - BotFather Commands Setup
=====================================================

Copy and paste these commands into BotFather when setting up your bot:

/setcommands

🎮 CORE GAME COMMANDS / دستورات اصلی بازی:

start - Start the bot and show welcome menu / شروع ربات و نمایش منوی خوش‌آمدگویی
menu - Show main game menu / نمایش منوی اصلی بازی
main - Show main game menu / نمایش منوی اصلی بازی
help - Show comprehensive help system / نمایش سیستم راهنمای جامع

⚔️ COMBAT COMMANDS / دستورات نبرد:

attack - Attack another player / حمله به بازیکن دیگر
weapons - Show weapons comparison and stats / نمایش مقایسه و آمار تسلیحات
battle_stats - Show your combat statistics / نمایش آمارهای نبرد شما

📊 INFORMATION COMMANDS / دستورات اطلاعاتی:

profile - Show detailed user profile / نمایش پروفایل تفصیلی کاربر
me - Show detailed user profile / نمایش پروفایل تفصیلی کاربر
status - Check your current game status / بررسی وضعیت فعلی بازی شما
stats - Show quick statistics overview / نمایش خلاصه سریع آمار
leaderboard - Show chat rankings / نمایش رده‌بندی چت
top - Show chat rankings / نمایش رده‌بندی چت
chat_stats - Show chat-wide statistics / نمایش آمارهای کلی چت

🛒 SHOP & INVENTORY COMMANDS / دستورات فروشگاه و موجودی:

shop - Browse available items for purchase / مرور آیتم‌های موجود برای خرید
store - Browse available items for purchase / مرور آیتم‌های موجود برای خرید
inventory - View your items and equipment / مشاهده آیتم‌ها و تجهیزات شما
inv - View your items and equipment / مشاهده آیتم‌ها و تجهیزات شما

⭐ PREMIUM COMMANDS / دستورات ویژه:

stars - Manage TG Stars and premium features / مدیریت ستاره‌های تلگرام و ویژگی‌های ویژه

⚙️ UTILITY COMMANDS / دستورات کاربردی:

language - Change bot language (English/Persian) / تغییر زبان ربات (انگلیسی/فارسی)
lang - Change bot language (English/Persian) / تغییر زبان ربات (انگلیسی/فارسی)

- Daily bonuses refresh every 24 hours
- Combat has cooldown periods to prevent spam

- همه دستورات فقط در چت‌های گروهی کار می‌کنند (به جز /start، /help، /language)
