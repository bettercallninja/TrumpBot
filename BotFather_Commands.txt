📋 TrumpBot v2.1.0 - BotFather Commands Setup
=====================================================

Copy and paste these commands into BotFather when setting up your bot:

/setcommands

🎮 CORE GAME COMMANDS / دستورات اصلی بازی:

start - Begin your battle journey and get main menu
menu - Show main game menu / نمایش منوی اصلی بازی
main - Show main game menu / نمایش منوی اصلی بازی
help - Get detailed help and instructions

⚔️ COMBAT COMMANDS / دستورات نبرد:

attack - Attack another player or select your weapon
weapons - Compare all available weapons
battle_stats - View your combat statistics

�️ DEFENSE COMMANDS / دستورات دفاعی:

shield - Activate a shield to block incoming attacks
defend - Activate intercept system to reduce hit chance
status - Check your HP, medals and active defenses

�📊 INFORMATION COMMANDS / دستورات اطلاعاتی:

profile - View your detailed profile
me - Show detailed user profile / نمایش پروفایل تفصیلی کاربر
stats - View your combat statistics and performance
leaderboard - See top players in the chat
top - Show chat rankings / نمایش رده‌بندی چت
chat_stats - Show chat-wide statistics / نمایش آمارهای کلی چت

🛒 SHOP & INVENTORY COMMANDS / دستورات فروشگاه و موجودی:

shop - Browse and buy weapons and items
store - Browse available items for purchase / مرور آیتم‌های موجود برای خرید
inventory - View your weapons and items
inv - View your items and equipment / مشاهده آیتم‌ها و تجهیزات شما
use - Use items like shields and boosters

⭐ PREMIUM COMMANDS / دستورات ویژه:

stars - View Telegram Stars balance and premium shop
bonus - Claim your daily bonus medals

⚙️ UTILITY COMMANDS / دستورات کاربردی:

language - Change bot language (English/Persian) / تغییر زبان ربات (انگلیسی/فارسی)
lang - Change bot language (English/Persian) / تغییر زبان ربات (انگلیسی/فارسی)
